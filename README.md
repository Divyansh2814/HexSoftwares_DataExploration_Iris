# Iris Dataset Data Exploration

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Iris dataset.

## Features

- Load the Iris dataset
- Explore the dataset
- Check missing values
- Summary statistics
- Histogram
- Box Plot
- Correlation Heatmap
- Pair Plot

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-Learn
